<h1>Bahar Azadi Coin Price Tracker</h1><br>

<p>A simple Python project that fetches the price of the Bahar Azadi gold coin from an external API and stores it in a MySQL database.</p>

<h2>Features</h2><br>

* Fetch coin price from API
* Store price in MySQL database
* Keep price history with timestamp

<h2>Tech Stack</h2><br>

* Python
* MySQL
* Requests

<h2>How it works</h2><br>

Fetches the latest price from API and saves it into the database on each run.

<h2>Database</h2><br>

Stores price value, change, timestamp, and date.
